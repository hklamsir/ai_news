# 智能波動率預測投資組合優化系統：MSVM框架

## 基本資訊
- **來源**：Nature Scientific Reports
- **標題**：Intelligent system for portfolio optimization for novel volatility forecasting using machine learning
- **作者**：伊朗、德國、馬來西亞研究團隊
- **發布**：2026年
- **DOI**：10.1038/s41598-026-42813-4

## 核心研究問題
現有投資組合優化模型存在兩大局限：
1. **只預測回報**，而非波動敏感性指標（如回撤比率），這對真實風險管理更重要
2. **缺乏可解釋性和泛化能力**，難以應用於不同國際市場

## 研究方法：MSVM框架

### 核心架構
- **多支援向量機（Multi-SVM）分類器**，結合RBF核函數
- 輸入：1999-2018年間40個國家的股票數據
- 特色：結合波動率特徵工程與傳統馬科維茨均值-方差模型

### 關鍵創新
1. **結構化_ratio特徵**：Sharpe比率、Sterling比率、Calmar比率
2. **信心加權融合機制**：動態調整分類器影響力
3. **跨市場一致性預處理**：Z-score標準化、異常值移除、類別不平衡處理
4. **70%-15%-15%訓練-驗證-測試分割**

### 實驗結果
- **準確率達97.5%**，超越傳統SVM和樸素貝葉斯分類器
- 涵蓋40個國際市場（發達+新興市場）
- 樣本期間：1999-2018年（20年數據）

## 技術特點

### 數據處理流程
1. 從Bloomberg數據庫獲取40國股票數據
2. 預處理：缺失值填補、Z-score標準化、IQR異常值移除
3. 特徵提取：最高/最低/平均股價、最大回撤、波動率、年度回報
4. 多SVM分類 + 信心加權融合

### 為何選擇SVM而非深度學習？
| 特性 | SVM | 深度學習 |
|------|-----|---------|
| 可解釋性 | 高 | 低 |
| 計算效率 | 高 | 低 |
| 小樣本表現 | 良好 | 需大量數據 |
| 不確定性量化 | 困難 | 困難 |
| 金融市場穩定性 | 優秀 | 不穩定 |

## 研究結論
- MSVM框架提供**可解釋、計算高效**的投資組合優化方案
- 在40個國際市場展現**良好泛化能力**
- 信心加權融合機制能動態適應市場體制變化
- 適合風險敏感型投資組合管理
