港大 DeepTutor 開源登場：將 PDF 教材轉化為互動個人化教學

香港大學數據科學實驗室推出開源 AI 學習助手 DeepTutor，採用多 Agent 協作架構，將 PDF 教材轉化為互動個人化教學，GitHub 累積逾 12,700 顆星。

🔑 核心功能解析

DeepTutor 的五大技術優勢與創新特點

🤖
多 Agent 協作架構
3 個專用 Agent 模組分工協作：逐步解題與評估、深度研究與引導學習、概念創意生成。各司其職，模擬真實補習過程。
📚
PDF 教材智能解析
一鍵上傳 PDF 教材，系統自動分析內容結構，轉化為互動式個人化教學。支援知識點拆解、概念視覺化及難度自適應調整。
✍️
自動模擬試卷生成
上傳一份考古題，系統自動分析風格與難度，生成風格一致的全新練習題。告別題庫不足的煩惱，輕鬆備考。
🔍
多源交叉驗證
整合 RAG 混合檢索、知識圖譜、即時網絡搜尋、學術論文資料庫及 Python 程式碼執行，回應時同步交叉驗證最新資料。

🤖 處理方式：AI 智能摘要生成 + 深度優化

DeepTutor 設計核心是讓學生真正理解知識，而非單純取得答案。與 ChatGPT 式問答截然不同，這是一個真正以學習者為中心的 AI 系統。

— HKUDS 官方定位

🎯 產品定位：「Agent 原生智能學習平台」

DeepTutor v1.0 於 2026 年 4 月 3 日正式推出，官方將其定位為「Agent 原生智能學習平台」。工具整合層涵蓋知識庫檢索、即時網絡搜尋、程式碼執行、深度推理、腦力激盪、論文搜尋及 PDF 檔案解析等核心功能。

🧠 三層 Agent 架構各司其職

DeepTutor 底層並非單一模型處理所有任務，而是由 3 個專用 Agent 模組分工協作：

  • 解題 Agent：負責逐步解題與評估，模仿真人補習老師的解題思路
  • 研究 Agent：負責深度研究與引導學習，幫助學生建立系統性知識框架
  • 創意 Agent：負責概念創意生成，以多元角度解釋抽象概念

🔮 應用場景與未來展望

對於複雜概念，DeepTutor 能將知識和演算法拆解為步驟，製作互動式逐步講解頁面，並結合雙迴圈推理架構，配合學生程度調整學習節奏。這個系統特別適合：

  • 大學生備考：自動分析 past paper 生成相似風格練習題
  • 在職進修：將專業論文轉化為個人化學習教材
  • 自學愛好者：建立個人知識圖譜，系統化掌握新領域
