NVIDIA Nemotron 3.5 ASR
純 CPU 即時轉錄 40 種語言

600M 參數填補本地 Agent 語音環節

📺 資料來源：KOC（電腦王阿達）

📅 發布日期：2026-06-20

🤖 處理方式：AI 智能摘要生成

🎯 核心觀點

NVIDIA 推出 600M 參數的 Nemotron 3.5 ASR 語音辨識模型，支援 40 種語言，純 CPU 即可即時轉錄。填補了本地 AI Agent 語音輸入環節的空白，讓完整本地 Agent pipeline 成為可能。

⚙️

技術架構：Cache-Aware FastConformer-RNNT

24 層 FastConformer 編碼器搭配 RNNT 解碼器，快取自注意力與卷積激活狀態，每個音訊幀只處理一次，計算量與延遲同步下降。

🌍

40 種語言，三層品質

透過 `target_lang` 指定語言（如 `zh-CN`），或設為 `auto` 自動偵測。自動偵測模式會在標點符號後輸出語言標籤，適合混合語言音訊。

⚡

延遲可調：att_context_size 參數

一個旋鈕控制速度與準確率平衡。串流場景達 258 倍即時處理，短音訊片段詞錯率不退化。開發者可根據場景（即時字幕 vs 批次轉錄）自由切換。

🔄

對比 Whisper：各有優勢

Nemotron 在串流場景全面勝出；Whisper 在嘈雜環境和帶口音音訊上穩健性更強。兩者定位互補，非取代關係。

📌 重點提示

Nemotron 3.5 ASR 以 OpenMDW-1.1 授權開源，模型已上架 HuggingFace，可用於商業用途。配合 Nemotron 3 Nano（語言模型）和 Magpie TTS（語音合成），從語音輸入到輸出的完整本地 Agent pipeline 將成為可能。

「對於正在搭建本地 AI Agent 的開發者而言，這意味著語音輸入不再是一個需要依賴雲端 API 的環節。」

— KOC 評測

🔮 業界展望

本地 AI Agent 的最後一塊拼圖正在到位。語音輸入、本地推理、語音輸出——三個環節都有對應的開源方案，且全部可在消費級硬體上運行。這將大幅推動對資料隱私有嚴格要求的行業應用，如醫療、法律、金融等領域的本地 AI Assistant。
