🎯 核心觀點

Anthropic 公佈 Claude Code 使用數據，發現非工程師用戶大幅增加，專業領域知識反而比過往更重要——越懂該領域的人，用一句話就能讓 AI 做更多有效工作。

📊

使用模式轉變

修 Bug 比例從 33% 降至 19%，操作軟體從 14% 升至 21%，寫作與資料分析從 10% 升至 20%。Claude Code 已不只是寫程式工具。

💰

任務價值上升 27%

建立新東西 +43%、操作軟體 +34%、修復任務 +32%。AI 代理正在取代部分白領重複性工作。

👥

非工程師快速成長

成長最快的非軟體職業是管理、銷售和法律。商業金融、藝術設計、媒體、生命科學等領域用戶均顯著增加。

🧠

專業知識決定產出效率

新手每 prompt 觸發 5 個動作輸出 600 字，專家觸發 12+ 個動作輸出 3,200 字。最大差距在新手與中階之間——只需對領域有一定理解，成功率就大幅提升。

📌 重點提示

Claude Code 用戶規劃決策占 70%，執行決策僅占 20%。現在只要能把需求和判斷標準講清楚，不用懂寫程式，全部交給 AI 執行即可。但專業知識不是被取代，而是變得更有價值。
