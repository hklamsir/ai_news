# GitHub Copilot全面轉向按Token計費！Coinbase：未來12-18個月 8成AI工作將由便宜99%的模型完成

## 📖 新聞導語
微軟旗下 GitHub Copilot 全面由「按請求次數收費」改為「按 Token 用量計費」，導致重度用戶帳單暴漲十倍。Coinbase 執行長 Brian Armstrong 預言，12 至 18 個月內 80% AI 工作負載將遷移至便宜 99% 的模型。

## 💡 核心內容

### 💰 GitHub Copilot 計費轉型
- **調整日期**：2026 年 6 月 1 日
- **改變**：從「按請求次數」改為「按 Token 用量計費」
- **用戶帳單飆升**：有開發者從每月 44.68 美元增至 754.29 美元
- **積分制**：1 積分 = 0.01 美元，訂閱層級決定基礎與彈性配額
- **用戶反彈**：批評此舉如同 Uber 先低價培養依賴再提價收割

### 💡 產品長預警
GitHub 產品長 Mario Rodriguez 今年 4 月已預警：
> 智能體 AI 興起後，一次簡短提問與耗時數小時的自主編程任務收費相同，平台一直在默默承擔飆升的推理成本。

### 📉 Coinbase 的破局策略
Coinbase 執行長 Brian Armstrong 預言：
- **80%** 的 AI 工作負載將遷移至成本低 **99%** 的模型
- **20%** 尖端任務（如科學突破、高階智能體編排）仍需前沿模型
- 比喻：「買頂配 MacBook 的是少數，而 AI 降價速度遠超摩爾定律」
- Coinbase 正推行「提示詞路由」，將請求動態分配至低成本模型

### 📊 開源小模型崛起
Hugging Face 執行長 Clement Delangue 引述史丹佛研究：
- 本地模型準確率從 2023 年的 **23.2%** 躍升至 **71.3%**
- 成本與能耗僅為前沿 API 的幾分之一
- 中國實驗室 DeepSeek V4 在程式設計基準測試媲美 Claude Opus，價格僅後者三十分之一

### ⚠️ 行業警示
- OpenAI 利潤率據悉接近 **-122%**，完全依賴外部資本補貼
- Gartner 分析師警告：Copilot 僅是開端，智慧體工作流程將進一步推高算力消耗

## 🔮 業界展望
AI 產業正從「資本輸血」走向「自我造血」。當補貼退潮，能源與算力將取代模型能力成為新瓶頸，定價地震預示著產業格局的重大重塑。
