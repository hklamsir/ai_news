# Qwen-AgentWorld：通用代理的語言世界模型

## 📖 新聞導語

阿里雲 Qwen 團隊正式開源 Qwen-AgentWorld-35B-A3B，這是首個覆蓋七個統一領域（MCP、Search、Terminal、SWE、Android、Web、OS）的原生語言世界模型，透過長期思鏈推理模擬代理環境，訓練數據超過 1000 萬條真實互補軌跡。

## 💡 核心內容

### 什麼是 Qwen-AgentWorld？

Qwen-AgentWorld 是一種「原生語言世界模型」，與過往將世界建模作為事後附加元件的做法不同，環境建模從 CPT 階段就已成為訓練目標。它透過三階段流程訓練：

- **CPT（持續預訓練）**：注入環境知識
- **SFT（監督微調）**：激活下一狀態預測推理
- **RL（強化學習）**：提升模擬保真度

### 七個統一領域

這是首個單一模型覆蓋七個代理交互領域的語言世界模型：

| 領域 | 說明 |
|------|------|
| MCP | 模型上下文協定 |
| Search | 搜尋引擎交互 |
| Terminal | 終端機操作 |
| SWE | 軟件工程任務 |
| Android | Android 設備控制 |
| Web | 網頁自動化 |
| OS | 操作系統操作 |

## 🔮 業界展望

Qwen-AgentWorld 的開源標誌著語言世界模型進入新階段。相較於傳統世界模型，世界建模從一開始就是訓練目標而非事後添加，這種「原生」方法在多個代理任務領域展現顯著優勢。模型權重和評測基準均已開源（Apache 2.0），為研究者和開發者提供了強大的工具。
