Anthropic 推出 Claude Opus 4.8
誠實，成為殺手級功能

新版 AI 模型錯誤率大降 4 倍，Dynamic Workflows 支援數百個子代理並行

📺 資料來源：ZDNET

📅 發布日期：2026-05-31

🤖 處理方式：AI 智能摘要生成

🎯 核心觀點

Anthropic 正式發布 Claude Opus 4.8，最大亮點是「誠實」。新版模型的錯誤率較前身降低 4 倍，更善於承認不確定性並主動修正錯誤。電子商務平臺 Shopify 的工程師搶先測試，形容其「判斷力明顯提升，會問正確的問題、捕捉自己的失誤」。

⚠️ 前代模型的困境

Opus 4.6 時期，Anthropic 的 AI 模型常出現誤解指令或交付錯誤結果的情況。用戶抱怨模型「自己創出答案」或「假裝理解」，影響實際工作效率。新版 Opus 4.8 針對這個痛點進行了根本性改善。

✨ Opus 4.8 三大核心升級

  • 誠實優先：主動承認不確定性，錯誤率降低 4 倍，不再假裝自己正確
  • Dynamic Workflows：可同時運行數百個並列子代理，自動驗證輸出結果，適用於大型程式碼遷移
  • Effort 分級：預設 High Effort 模式以相同 Token 用量達到更好效能，亦登陸 Claude.ai 和 Claude Code

✨ 關鍵洞察：當 AI 代理數量爆發，「誠實」決定成敗

當 Claude 需要同時協調數百個子代理時，人力監督根本無法跟上。如果每個子代理都可能「假裝正確」，整個系統將失去可靠性。Opus 4.8 的誠實機制，正是為這個場景而設計。
