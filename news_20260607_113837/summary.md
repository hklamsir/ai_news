# OpenAI Lockdown Mode：保護敏感數據防範提示注入攻擊

## 📖 新聞導語

OpenAI 推出全新 **Lockdown Mode** 安全功能，專為處理敏感數據的個人和企業用戶設計。此模式透過限制 ChatGPT 連接網絡或外部服務的能力，降低提示注入（Prompt Injection）攻擊導致數據外洩的風險。

## 💡 核心內容

### 什麼是 Lockdown Mode？

Lockdown Mode 是一個可選的高級安全設定，能限制 OpenAI 產品中許多可連接網絡或外部服務的工具和功能。其設計目的是透過限制向外發送的網絡請求，來減少提示注入攻擊導致數據外洩的風險——代價是犧牲部分實用功能。

### Lockdown Mode 會停用哪些功能？

啟用後，以下功能將被限制或停用：

- **即時網頁瀏覽**：僅可存取快取內容，搜尋結果可能有限或過時
- **圖片支援**：無法在一般回覆中顯示圖片或從網絡檢索圖片（仍可上傳圖片及生成圖片）
- **深度研究**：停用
- **代理模式**：停用
- **Canvas 網絡**：無法批准 Canvas 生成的代碼訪問網絡
- **檔案下載**：無法下載檔案進行數據分析

### Lockdown Mode 不影響什麼？

- 記憶功能
- 檔案上傳
- 對話分享
- 模型訓練設定（可另行配置）
- Codex 的網絡訪問

### 重要限制

Lockdown Mode **無法完全阻止**提示注入攻擊。例如，藏在快取網頁內容或上傳檔案中的惡意指令，仍可能影響 ChatGPT 的行為或準確性。OpenAI 強調，此模式不是為所有人設計，適用於處理敏感數據且希望獲得嚴格保護的個人或組織。

##🔮 業界展望

提示注入是一個前沿且具挑戰性的研究問題，攻擊者可能在網頁或其他內容來源中隱藏惡意指令。Lockdown Mode 的推出顯示 OpenAI 持續強化多層安全系統，但專家指出此模式不能保證百分百防護，風險仍可能存在於已啟用的應用程式、功能組合或新發現的攻擊技術中。
