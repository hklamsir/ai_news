# 阿里，把 GLM-5.2 免費了

## 📖 新聞導語

中國 AI 公司 Z.ai（前稱智譜 AI）發布了 GLM-5.2 开源模型，在多項基準測試中與 OpenAI 和 Anthropic 的模型並駕齊驅，但成本僅為美國頂級封閉模型的六分之一，震動全球 AI 業界。

## 💡 核心內容

### GLM-5.2 開源模型的崛起

GLM-5.2 是一個「开 weight」模型，意即任何人都可以下載、運行和修改它。這種開放性使其對成本敏感的企業和開發者具有極大吸引力。

根據 Artificial Analysis 的大型語言模型排行榜，GLM-5.2 目前排名第五；在 Code Arena 的前端編碼排名中位居第二，而成本僅為 Claude 和 GPT 系列等封閉模型的六分之一。

### 業界大佬點名肯定

美國前 AI 沙皇 David Sacks 在 All-In Podcast 中表示：「我們現在有一個中國的开 weight 模型，與 OpenAI 和 Anthropic 目前的模型一樣好。」他還指出 GLM-5.2 「僅略低於 Anthropic 的 Opus 4.8，並與 OpenAI 的 GPT 5.5 並駕齊驅。」

### ZCode 同步發布

Z.ai 同步推出了 ZCode，這是一個免費的桌面開發環境，支援 macOS、Windows 和 Linux。官方描述其為「Agentic Development Environment」，專為 GLM-5.2 大型語言模型打造，直接挑戰 Cursor、Claude Code 和 GitHub Copilot 的市場地位。

### 價格戰開打

GLM-5.2 的成功表明，企業對封閉源 AI 的高昂且不可預測的成本越來越敏感。隨著代理型 AI 工具消耗更多 tokens，企業成本不斷攀升，開源替代方案的吸引力正在增強。

## 🔮 業界展望

GLM-5.2 的出現將加劇 AI 領域的競爭，特別是在開源模型市場。這一發展可能迫使美國 AI 公司在定價上做出讓步，同時也引發了關於中國是否正在趕超美國 AI 能力的廣泛討論。隨著更多開源選項出現，AI 技術的普及門檻將進一步降低。
