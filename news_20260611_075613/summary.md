# Decart 世界模型可模擬數小時超真實駕駛場景——但有條件

## 📖 新聞導語

AI 新創公司 Decart 發布最新世界模型 **Oasis 3**，能即時生成超真實駕駛環境，目標客戶包括自動駕駛公司及開發者社群，並獲豐田、Adobe、eBay 等策略投資。

## 💡 核心內容

### 🚗 Oasis 3 核心能力
Oasis 3 是互動式世界模型，可即時生成多鏡頭（前方 + 兩側）超真實駕駛環境。開發者可用文字提示生成場景，並可互動數小時，適合訓練及測試自動駕駛系統。相較 Google Genie 3 或 World Labs Marble，Oasis 3 是目前單一文字提示下最超真實的選擇。

### ⚠️ 主要限制
長時間使用後模型品質會明顯下降（degrades significantly），這是目前的主要缺點。

### 💰 巨額融資
Decart 兩年前成立，最近數週完成 **3 億美元（約新台幣 97 億元）**融資，估值接近 **40 億美元**。投資者包括豐田（Toyota）、Adobe、eBay、Nvidia 等策略夥伴，全為潛在客戶。

### 🎯 商業策略
不同於競爭對手提供有限展示或研究預覽，Decart 從第一天就開放 API 存取，目標是建立類似 OpenAI 的開發者生態系統。初期瞄準需要大規模模擬罕見駕駛場景的自動駕駛公司，長期計畫扩展至機器人及實體 AI 應用。

### 🏔️ 競爭環境
世界模型戰場日益擁擠：Google 去年發布 Genie 3 研究預覽版、Fei-Fei Li 的 World Labs 推出商用 Marble、Luma 與 Runway 等影片生成新創也进军世界模型領域。

## 🔮 業界展望

Decart 的 Oasis 3 代表世界模型在自動駕駛模擬領域的重要進展。3 億美元融資顯示資本市場對此技術的高度興趣。隨著 API 開放，生態系統建設將是成敗關鍵——誰能像 OpenAI 一樣吸引開發者，誰就能主導這個新興領域。
