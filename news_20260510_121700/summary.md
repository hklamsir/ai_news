# OpenAI 推出 Codex Chrome 插件：進軍瀏覽器開發工作

## 📖 新聞導語

**來源**：Engadget　**發布日期**：2026-05-10

OpenAI 推出 Codex 的 Chrome 擴充功能，讓這款 AI 程式碼助理可在瀏覽器環境中測試網頁應用、跨分頁收集上下文、同時使用 Chrome 開發者工具。此插件目標是吸引程式設計師以外的更多專業用戶。Codex 於今年 2 月以 macOS 應用程式推出，現在也支援 Windows，預計未來將整合 ChatGPT 與 Atlas 網頁瀏覽器。

## 💡 核心內容

### Chrome 插件新功能

Codex 的新瀏覽器功能包括：

- **測試網頁應用**：直接在瀏覽器中運行和測試 Web 應用程式
- **跨分頁收集上下文**：從多個打開的分頁中獲取相關資訊
- **並行使用 Chrome DevTools**：同時操作開者工具而不影響用戶瀏覽體驗
- **保持結果有組織**：自動整理測試結果，不會接管整個瀏覽器

### 目標用戶群擴展

OpenAI 的策略是將 Codex 從純粹的開發者工具轉變為更廣泛的專業人士助手。由於許多計算任務都在瀏覽器中進行，因此非技術背景的專業人士（如產品經理、設計師、數據分析師）也能從中受益。

### 發展時間線

- **2026 年 2 月**：Codex 以 macOS 應用程式形式首次亮相
- **2026 年 4 月**：新增更多功能
- **未來計劃**：將 Codex 與 ChatGPT 聊天機器人及 Atlas 網頁瀏覽器整合為統一應用程式

### 跨平台支援

Chrome 擴充功能支援 Windows 和 Mac 系統，進一步擴大了 Codex 的用戶基礎。
